# Little Lemon - Android App

This project is a final assignment completed as part of a Coursera Android development course.

The application is a simple Android app written in Kotlin that demonstrates a basic authentication flow and navigation between screens.

## Features

- User registration
- Login and logout
- Home screen
- Screen-to-screen navigation
- Clean and structured UI

## Tech Stack

- Kotlin
- Jetpack Compose
- Android Navigation
- MVVM architecture
- Material Design
